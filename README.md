# sse-test

# Install node modules
1. Navigate to sse-client - `cd sse-client`
2. Install node-modules - `npm i`

## To start the server
1. Navigate to sse-server - `cd sse-server`
2. Start the server - `node server.js`

## To Start the client
1. Navigate to sse-client - `cd sse-client`
2. Start the client - `npm start`
